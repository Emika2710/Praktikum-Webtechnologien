# Praktikum-Webtechnologien
## Install

### Windows Xampp

1) Backup old htdocs
1) Symlink project to htdocs

```cmd
rem run in cmd.exe
cd C:\xampp
ren htdocs htdocs.backup
mklink /J C:\xampp\htdocs [Path to]\Praktikum
```

#### run

1) Open Xampp-control-panel
1) start apache
1) open the localhost (127.0.0.1) in the browser
1) profit
